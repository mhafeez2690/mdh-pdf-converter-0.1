
# MDH PDF Converter App

This Streamlit app allows users to convert PDF files into Word (.docx) or Excel (.xlsx) formats.

## Features
- PDF to Word conversion
- PDF to Excel conversion with table detection
- OCR fallback for scanned PDFs
- MDH branding and help section
- Version history

## How to Use
1. Upload a PDF file.
2. Choose conversion type.
3. Click Convert and download the result.
